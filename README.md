# ECC3479 Project

## Research Question

How are changes in international student enrolments associated with rental vacancy rates in Melbourne?

This project examines whether international student demand is descriptively associated with Melbourne rental vacancy rates. The analysis is exploratory and descriptive, not causal.

## Repository Structure

- `README.md`: project overview, repository structure, software information, and reproducibility instructions
- `code/`: Python scripts and notebooks used in the project
  - `clean_data.py`: cleaning and validation script that converts the raw integrated quarterly dataset into the final analysis-ready dataset
  - `eda.ipynb`: exploratory data analysis using the cleaned quarterly dataset
  - `analysis.ipynb`: primary descriptive econometric analysis
  - `robustness_analysis.ipynb`: robustness analysis for the primary econometric result
- `data/raw/`: raw integrated dataset and data source documentation
  - `quarterly_student_vacancy_2020_2025.csv`: raw integrated quarterly dataset used as the input to the cleaning script
  - `README.md`: documentation on data sources, access, construction, and limitations
- `data/clean/`: cleaned analysis-ready dataset and codebook
  - `clean_data.csv`: final cleaned dataset generated by `code/clean_data.py`
  - `codebook.md`: variable documentation for the cleaned dataset
- `docs/`: notes and supporting documentation
- `outputs/`: generated tables and other reproducible outputs
  - `outputs/tables/regression_table_quarterly.csv`: primary regression table
  - `outputs/tables/robustness_table.csv`: robustness analysis table
  - `outputs/tables/robustness_table.html`: HTML version of the robustness analysis table
- `requirements.txt`: Python package requirements
