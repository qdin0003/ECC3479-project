# Clean Data Codebook

This codebook describes the variables in `data/clean/clean_data.csv`.

The cleaned dataset is generated by `code/clean_data.py` from the raw integrated quarterly dataset stored in:

```text
data/raw/quarterly_student_vacancy_2020_2025.csv
```

The dataset contains quarterly observations from 2020 Q1 to 2025 Q4.

## Dataset structure

- Number of observations: 24
- Number of variables: 13
- Unit of observation: quarter
- Time period: 2020 Q1 to 2025 Q4
- Student data geography: Victoria
- Vacancy data geography: Melbourne

## Variables

| Variable | Description |
|---|---|
| `year` | Calendar year of the observation. |
| `quarter` | Quarter label, from Q1 to Q4. Kept as a categorical variable for seasonal controls and quarter fixed effects. |
| `quarter_end_month` | Quarter-ending month label, such as Mar, Jun, Sep, or Dec. |
| `quarter_end_date` | Date corresponding to the end of the quarter. |
| `vic_ytd_enrolments` | Victorian year-to-date international student enrolments. This is the main explanatory variable in the econometric analysis. |
| `vic_ytd_commencements` | Victorian year-to-date international student commencements. Used as an additional student-demand control in robustness checks. |
| `estimated_melbourne_vacancies` | Estimated number of rental vacancies in Melbourne. |
| `estimated_melbourne_vacancy_rate_pct` | Estimated Melbourne rental vacancy rate, measured in percentage points. This is the main outcome variable. |
| `student_geography` | Geographic coverage of the student data. |
| `vacancy_geography` | Geographic coverage of the vacancy data. |
| `notes` | Notes on data construction or interpretation. |
| `quarter_num` | Numeric version of the quarter variable, where Q1 = 1, Q2 = 2, Q3 = 3, and Q4 = 4. Used for sorting and validation. |
| `quarter_end_month_num` | Numeric version of the quarter-ending month, where Mar = 3, Jun = 6, Sep = 9, and Dec = 12. Used for validation and chronological checks. |

## Main analysis variables

The main outcome variable is:

```text
estimated_melbourne_vacancy_rate_pct
```

The main explanatory variable is:

```text
vic_ytd_enrolments
```

An additional student-demand control used in robustness checks is:

```text
vic_ytd_commencements
```

## Data construction notes

The international student variables are measured at the Victoria level. The rental vacancy variables refer to Melbourne. This geographic mismatch is a limitation of the analysis and is discussed in the project README and supporting documentation.

The Melbourne vacancy-rate values are estimated from public SQM Research vacancy-rate charts because the underlying SQM table data require purchase. As a result, the vacancy variables should be interpreted as approximate rather than exact administrative data.

## Analysis interpretation

The analysis is descriptive rather than causal. The project examines whether Victorian international student enrolments are conditionally associated with the estimated Melbourne rental vacancy rate in the available quarterly data.

The dataset is a quarterly aggregate time-series dataset, not an LGA-level panel dataset. Therefore, area fixed effects and clustered standard errors by LGA are not used in the final robustness analysis.
